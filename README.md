# 股市猎人测验 📈🗡️
一个有趣又犀利的股市投资能力测试网页，帮你判断：你是持镰刀的猎人，还是待割的韭菜？

## ✨ 项目介绍
本测试包含 **15 道专业股市问题**，覆盖：
- 财务分析
- 技术分析
- 宏观经济
- 风险管理
- 交易心态
- 资金管理
- 核心竞争力

通过答题自动计算得分，生成你的**股市生存报告**。

## 🎯 测试结果等级
- 🗡️ **持镰刀的猎人**：具备股市生存能力
- 🎭 **徘徊的学徒**：有基础但仍需学习
- 🐑 **待宰的羔羊**：知识严重不足
- 🌱 **纯种韭菜**：不适合直接入市

## 🎮 使用方法
1. 打开在线链接
2. 点击「开始测验」
3. 每题选择最符合你的答案
4. 完成后查看专属股市分析报告

## ⚠️ 重要提示
本测验仅供学习与自我评估使用，不构成任何投资建议。
